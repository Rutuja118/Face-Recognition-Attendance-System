Face Recognition Attendance System

A smart attendance system that uses AI-based Face Recognition to automatically mark attendance using a webcam.

🚀 Features

Real-time face detection

Automatic attendance marking

Stores date and time

User-friendly GUI using Tkinter

Attendance saved in Excel/CSV

MySQL database support


🛠 Technologies Used

Python

OpenCV

Face Recognition Library

NumPy

Pandas

Tkinter

MySQL

OpenPyXL

▶️ How to Run

1. Install Python


2. Install required libraries



pip install opencv-python face_recognition numpy pandas mysql-connector-python openpyxl

3. Run the project



python main.py

📸 Working

The webcam captures faces

The system recognizes registered users

Attendance is automatically stored with date and time
